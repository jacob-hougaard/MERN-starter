# Mern starter

## About this repository

This is a starter template for a Mongo, Express, React, Node application. The application comes with typescript.

## Starting the app

To run the backend application you need to type in the following command

```npm install```
Along with
```npm run start:dev```


For the client, run ```npm install``` followed by ``npm start``

To run the database run the following command in the ```/database``` directory:

```docker compose up```
